# IACC: Cross Illumination-Aware and Colour Correction for Underwater Images Under Mixed Natural and Artificial Lighting
# Underwater Image Enhancement using IACC

## Overview
This project implements underwater image enhancement using:
- Histogram Equalization
- CLAHE
- CLCC (Deep Learning Model)

## Features
- Multi-method comparison
- PSNR & SSIM evaluation
- Visualization pipeline

## Note
The deep model (CLCC) can be trained using the provided training pipeline.
